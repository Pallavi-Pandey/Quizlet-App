# Quizlet Clone

A functional clone of the popular Quizlet platform, designed for creating and studying digital flashcards and quizzes.

## Key Features
- **Flashcard Creation**: Interactive interface for inputting questions and answers.
- **Study Modes**: Multiple ways to review and test your knowledge.
- **Modern UI**: Clean and intuitive design optimized for effective learning.

## Getting Started
Each project version contains specific instructions for setting up the frontend and backend.
```bash
npm install
npm run start
```
Explore the codebase to see how state management and user interactions are handled in this educational app.
